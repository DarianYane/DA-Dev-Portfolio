What is Fast Reading?

Fast Reading allows you to format a text to make it read faster.

This application allows you to upload a file and optimize it for on-screen reading, or download it.

How did I come up with the idea?

One day I was reading this note and I found it a very interesting idea.

At first I thought that someday I would develop an application for cell phones to generate files in this format.

Some time later, when I started thinking about the projects I would include in my portfolio, I remembered the idea. It seemed like an excellent project for a junior developer, which would be an original idea, and at the same time, it would be a challenge for my skills.

How does it work?

The first thing to do is to upload a file from your computer.

Once the file is uploaded, Fast Reading identifies all spaces and punctuation, and boldfaces the first two letters of the next word.

What kind of files can I optimize?

For now, you can only optimize PDF files.

Is this the final version of the application?

No. In the future I will continue to optimize it to accept other file types, identify titles and page numbers, etcetera.

How do I convert a file?

Click to select the file you want to optimize.

Then click on the "Upload" button.

The file will be optimized and you will be able to read it on screen, or download it in PDF format.
