What is SQL Cleaning Data Project?

In this project you receive an excel file with information that needs to be cleaned (data with errors, NULL, duplicates, etc).

Through the use of SQL, the data is cleaned so that later it can be processed correctly.

Scope

Take a list of property sales and clean up the information using SQL code.

What improvements have been made?

During the optimization process, the following improvements were made:

  • Addition of new fields
  • Complete missing data (NULL)
  • Separation of data in different columns
  • Correct data format
  • Grouping of categories
  • Duplicate removal
  • Elimination of unnecessary columns

What are the steps in the cleaning process?

  1. Download the dataset
  2. Run SQL Server Management Studio and connect to server
  3. Create a new database named SQLCleaningDataProject
  4. Right click / tasks / import data
  5. In the drop-down menu, select Microsoft Excel
  6. Browse
  7. Select the Excel file
  8. Next
  9. Select "Microsoft OLE DB Provider for SQL Server"
  10. Next
  11. Select Sheet1
  12. Next
  13. Finish
  14. Finish
  15. Close
  16. Go to SQL Server Management Studio
  17. Select the database created
  18. Tables

    19. Preliminary analysis
  19. Rename the table to CityHouses if necessary
  20. Right click / Select top 1000 rows (to see what's in the table)

    21. Optimizing the data
  21. Delete the ID column (using ALTER TABLE and DROP COLUMN)
  22. Some PropertyAddress are NULL. If your ParceID repeats with another SaleDate, and that repeated ParcelID has an address, replacethe nulls with the corresponding PropertyAddress. (using AS, JOIN, ON, AND, <>, WHERE, NULL, ISNULL, UPDATE, and SET)
  23. Separate the property address data (using ALTER TABLE, ADD, SUBSTRING, CHARINDEX, AS, LEN, UPDATE, SET, and DROP COLUMN) into:
    • Address (Street + Number)
    • City
  24. Change the format of the sale date (using CONVERT, ALTER TABLE, ADD, UPDATE SET and DROP COLUMN)
  25. Separate the address data of the property owner (using ALTER TABLE, ADD, PARSENAME, REPLACE, UPDATE, SET, and DROP COLUMN) into:
    • Address (Street + Number)
    • City
    • State
  26. Replace "Y" with "Yes" and "N" with "No" (ysando GROUP BY, UPDATE, SET, CASE, WHEN, THEN and ELSE)
  27. Remove duplicates (using WITH (CTE) AS, ROW_NUMBER, OVER, PARTITION BY, ORDER BY, DELETE and WHERE).
