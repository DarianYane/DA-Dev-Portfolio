Real API Project - Employee Management System

Overview

The Employee Management System is a web-based application that allows companies to manage their employee data, departments, and hiring activities. It provides functionality to add, edit, and delete employees, as well as track department information and hiring metrics. The system is built using Django, a Python web framework, and utilizes a SQLite database for data storage.

You can access the API graphical interface here.

What is the aim of this work?

The objective of this job is to demonstrate my API and SQL skills to fill a Data Engineer position.

Why did I do this job?

This job was requested by a multinational company as a technical challenge to fill a Senior Data Engineer position.

The challenge consisted of a mandatory and an optional part. Among the mandatory items, it was requested to generate an API to perform massive data upload (more than 1000 lines) through CSV files, and to perform 2 complex queries.

Features

  • Employee management: Add, edit, and delete employee records with details such as name, job, department, and hire date.
  • Same features for departments and jobs
  • Uploading of new data can be done via API forms or by sending CSV files.
  • Hiring metrics: Generate reports on employee hiring activity, such as the number of employees hired by department and quarter.
  • API integration: Expose API endpoints to interact with the system programmatically.
  • Data can be retrieved in 3 different formats: JSON through the API GUI, pure JSON, or tables within an HTML.

Database Schema

The database schema consists of two main tables:

  • Employee: Stores employee information such as name, job, department, id and hire date.
  • Department: Stores department details including department name and id.
  • Job: Stores job details including job name and id.

API Endpoints

The following API endpoints are available for interacting with the system:

Endpoint Description Request Method
/api/hired-employees Retrieve a list of all employees or create a new employee record. GET, POST, HEAD, OPTIONS
/api/hired-employees/{id} Retrieve, update, or delete an employee record with the specified ID. GET, PUT, PATCH, DELETE, HEAD, OPTIONS
/api/departments Retrieve a list of all departments or create a new department record. GET, POST, HEAD, OPTIONS
/api/departments/{id} Retrieve, update, or delete a department record with the specified ID. GET, PUT, PATCH, DELETE, HEAD, OPTIONS
/api/jobs Retrieve a list of all jobs or create a new job record. GET, POST, HEAD, OPTIONS
/api/jobs/{id} Retrieve, update, or delete a jobs record with the specified ID. GET, PUT, PATCH, DELETE, HEAD, OPTIONS
