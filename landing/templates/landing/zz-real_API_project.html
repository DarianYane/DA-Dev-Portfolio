Real API - Employee Management System

Overview

The Employee Management System is a web-based application that allows companies to manage their employee data, departments, and hiring activities. It provides functionality to add, edit, and delete employees, as well as track department information and hiring metrics. The system is built using Django, a Python web framework, and utilizes a PostgreSQL database for data storage.

What is the aim of this work?

The aim of this work is to determine which are the characteristics of people who consider themselves happier and less happy than the average.

Why did I do this job?

I did this work to demonstrate my skills as a data analyst and in the use of Tableau. It can be found on my Github and in my portfolio at darianyane.com.

Features

  • Employee management: Add, edit, and delete employee records with details such as name, job, department, and hire date.
  • Department management: View and update department information, including department name and location.
  • Hiring metrics: Generate reports on employee hiring activity, such as the number of employees hired by department and quarter.
  • API integration: Expose API endpoints to interact with the system programmatically.
  • Data validation: Implement data validation rules to ensure the accuracy and integrity of the stored information.
  • Error handling and logging: Handle and log errors to facilitate troubleshooting and system maintenance.
  • Automated tests: Develop automated tests to verify the functionality and stability of the system.

Database Schema

The database schema consists of two main tables:

  • Employee: Stores employee information such as name, job, department, and hire date.
  • Department: Stores department details including department name and location.

API Endpoints

The following API endpoints are available for interacting with the system:

Endpoint Description Request Method
/api/employees Retrieve a list of all employees or create a new employee record. GET, POST
/api/employees/{id} Retrieve, update, or delete an employee record with the specified ID. GET, PUT, DELETE
/api/departments Retrieve a list of all departments or create a new department record. GET, POST
/api/departments/{id} Retrieve, update, or delete a department record with the specified ID. GET, PUT, DELETE
